Item Handover Management

1 Receive Item from Finder

When the finder arrives with the item:
  • Verify they have the item matching the description
  • Ask them for their pickup code (they received it via email)
  • Enter the code below to confirm receipt
  • Store the item securely until the claimer arrives

2 Hand Over Item to Claimer

When the claimer arrives to collect:
  • Ask for their pickup code (same code as the finder)
  • Verify their ID matches the claim information
  • Ensure the item was already received from the finder (Step 1 complete)
  • Enter the code below to complete the handover
